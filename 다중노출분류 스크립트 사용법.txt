python classify.py --imgtype CR2 --filename 12 --time_long 8 --start_folder_number 1

imgtype : 분류할 이미지 확장자(CR2나, JPG)
filename : 사진기에 찍힌 이미지 파일 확장자까지 포함해서 파일이름길이
예) IMG_0321.CR2 : 12자
     DSC02205.ARW : 12자
time_long : 인터벌 릴리즈로 사진 찍었을때, 한타임에 찍은 사진의 간격수 조정. 예를들어 10초동안 사진을 찍었다면 time_long 을 10으로 두면 된다.
start_folder_number : 시작하는 폴더 번호를 지정 가능

사용하는 경우
월식찍을때 예를들어서
3분마다 8초씩 촬영 들어간다고 가정하면

첫번째 묶음 : 03:00에 장당 1/320초로 8초동안 20장 촬영됨
두번째 묶음 : 03:03에 장당 1/160초로 8초동안 18장 촬영됨
.....

n번째 묶음 : 03시 + 03*(n-1)분에 장당 1초로 8초동안 5장 촬영
n+1번째 묶음 : 03시 +03*n 분에 장당 4초로 8초동안 2장 촬영됨


이렇게 찍었다고 합시다

그럼 파일 형식(CR2냐 JPG냐)하고, 파일 이름 길이하고(사진기 뭐쓰냐에 따라 파일이름 형식이 달라지니까, 그리고 연사하는 시간을 알면
맨 윗줄의 방식으로 명령프롬포트에 입력하면, CR2파일들을 분류해서, 폴더별로 복사해줍니다. 폴더 시작번호는 1번이고

첫번째 묶음 20장은 폴더 (1)에
두번째 묶음 18장은 폴더 (2)에
n번째 묶음 5장은 폴더 (n)에
n+1번째 묶음 2장은 폴더 (n+1)에

**classify는 getmtime 을 사용해 파일 '만든날짜'를 기준으로 분류하고
  classify_exifread는 python exifread 패키지가 필요하며, 이때 이미지 EXIF Header의 OriginDateTime을 기준으로 분류합니다.
